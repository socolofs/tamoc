tamoc.particle_size_models.PureJet.update_properties

tamoc.particle_size_models.PureJet.update_properties#

PureJet.update_properties(rho_p, mu_p, sigma_p, rho, mu, fp_type)[source]#

Set the thermodynamic properties of the release and receiving fluids

Parameters

  • rho_p (float) – Density of the gas phase released from the jet (kg/m^3)

  • mu_p (float) – Dynamic viscosity of the gas phase released from the jet (Pa s)

  • sigma_p (float) – Interfacial tension between the gas phase released from the jet and the continuous-phase receiving fluid (N/m)

  • rho (float) – Density of the continuous-phase receiving fluid (kg/m^3)

  • mu (float) – Dynamic viscosity of the continuous-phase receiving fluid (Pa s)

  • fp_type (int, default 1) – Phase of the released fluid; 0 = gas, 1 = liquid.
