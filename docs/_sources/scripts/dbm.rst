###############################
Discrete Particle Model Scripts
###############################

The ``dbm`` module in ``tamoc`` provides equations of state for various fluids and empirical relations for solid and fluid particle properties.  The following scripts demonstrate how to use these tools for various fluids.  To see the actual source code, consult the ``.py`` files provided in the ``./bin`` directory of ``tamoc``.
