###########################
Single Bubble Model Scripts
###########################

The ``single_bubble_model`` in ``tamoc`` simulates the Lagrangian pathway of a single bubble, droplet, or particle rising through the ocean water column.  The following scripts demonstrate how to set up simulations, save data, and interact with the results.  To see the actual source code, consult the ``.py`` files provided in the ``./bin`` directory of ``tamoc``.
