##############################
Stratified Plume Model Scripts
##############################

The ``stratified_plume_model`` simulates a multiphase plume in the absence of ambient currents, but considering ambient density stratification.  The following example files demonstrate how to use the model.  To see the actual source code, consult the ``.py`` files provided in the ``./bin`` directory of ``tamoc``.
