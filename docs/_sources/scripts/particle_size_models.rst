###########################
Particle Size Model Scripts
###########################

The ``particle_size_models`` module in ``tamoc`` provides tools to estimate bubble and droplet size distributions for jet releases using different empirical formulas available from the literature.  The following scripts demonstrate a single-phase release and a multi-phase release.  To see the actual source code, consult the ``.py`` files provided in the ``./bin`` directory of ``tamoc``.
